사용 기술

  • Overall

    1. 사용자와 소통하는 인터렉션을 좋아합니다.
    2. 업무에 필요하다면 낯선 라이브러리이더라도 적극적으로 탐색하여 최적의 결과를 낼 수 있도록 노력합니다.
    3. 솔직하며 직설적인 커뮤니케이션을 선호합니다.

    Communication

    1. 직위나 포지션에 관계없이 적극적으로 생각을 피력합니다.
    2. 지적 겸손함을 유지하기 위해 노력합니다.
    3. 커뮤니케이션은 적은 것보다는 많은게 좋다고 생각합니다.

    HTML / CSS / ANIMATION

    1. 시맨틱 마크업을 준수합니다.
    2. 다양한 플랫폼 및 브라우저를 지원할 수 있습니다.
    3. Sass 등 CSS Preprocessor로 유틸을 만들어 유지보수가 되는 마크업을 개발합니다.
    4. BEM 등의 CSS 방법론을 적용할 수 있습니다.
    5. CSS cubic-bezier 관련 지식으로 SVG 애니메이션을 구현합니다.

    JavaScript / React

    1. ES6 + 자바스크립트 문법을 활용합니다.
    2. React는 클래스형 뿐만 아니라 함수형 컴포넌트로 개발하며 Custom Hook을 만들어 사용합니다.
    3. Redux로 전역 상태 관리를 하며 flux구조를 이해하고 있습니다.

    Tooling

    1. Git을 활용한 형상관리를 하며 git flow로 브랜치 배드 전략을 수립합니다.
    2. Visual Studio Code, Eclipse 등의 개발 툴을 활용합니다.
